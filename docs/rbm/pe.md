# PE系统

通过PE模式，我们可以对机器进行系统修复/重置系统密码/找回数据等操作。PE模式是一种独立的系统环境，可以在不启动操作系统的情况下对硬盘进行操作。

## Linux系统

首先我们需要雨云后台启动控制台，等待启动完成后，按步骤操作，选择对应的镜像，然后重启机器即可进入PE模式。
