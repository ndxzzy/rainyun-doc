# 积分商城

积累够多积分之后，可以在[商城](https://app.rainyun.com/account/reward/store)兑换需要的产品。<br/>
积分目前可以兑换免费云服务器，域名，优惠券。
:::tip
积分还可以为产品进行等价续期<br/>
积分可以直接提现 (2000积分=1元)
:::
