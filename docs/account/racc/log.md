# 账号变动日志

## 账号变动日志

如果你认为你的账号被别人登陆，又或者需要查询自己的操作变动，可以在[账号变动日志](https://app.rainyun.com/account/settings/logs)，查询账号近期的所有变动操作。

## 如何查询
访问`https://app.rainyun.com/account/settings/logs`
后，根据页面提示，选择需要查询的日期范围，即可查询到相应的操作记录，如果有疑问可以发工单咨询客服。
